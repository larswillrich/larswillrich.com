# larswillrich.com

Personal website for Lars Willrich — Data & AI Principal Consultant based in Zurich.

## Tech Stack

- Static HTML/CSS/JS
- Nginx (Alpine) via Docker
- Contact form via [Formsubmit.co](https://formsubmit.co)

## Run Locally

```bash
docker compose up --build
```

The site will be available at [http://localhost:8080](http://localhost:8080).
